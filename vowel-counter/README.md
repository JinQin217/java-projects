# Vowel Counter (Java)

## Overview
A Java program that counts the occurrences of vowels in a user-input sentence and displays the results in a structured format.

## Features
- Counts occurrences of vowels (a, e, i, o, u)
- Displays total number of vowels
- Uses ArrayList for data storage
- Handles user input dynamically

## Example
Enter a sentence:
hello world<br>
Output:<br>
[('u', 0), ('o', 2), ('i', 0), ('e', 1), ('a', 0)]<br>
There are 3 vowels in this sentence


## Motivation
This project was built to practice string manipulation, iteration, and basic data structures in Java.

## Concepts Used
- String processing
- ArrayList
- Loops and conditionals

## How to Run
1. Compile:
   javac VowelCount.java
2. Run:
   java VowelCount
