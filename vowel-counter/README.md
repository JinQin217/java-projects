# Vowel Counter (Java)

## Overview
A program that counts the number of vowels in a user-input sentence.

## Features
- Counts occurrences of a, e, i, o, u
- Outputs total vowel count
- Displays formatted results

## Concepts Used
- String processing
- Arrays / ArrayList
- Loops and conditionals

## How to Run
1. Compile:
   javac VowelCount.java
2. Run:
   java VowelCount
