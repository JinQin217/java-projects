# Rectangle Class (Java)

## Overview
A simple Java class representing a rectangle, demonstrating object-oriented design and basic geometric calculations.

## Features
- Calculates area
- Calculates perimeter
- Supports default and parameterized constructors
- Provides getter methods for dimensions

## Example Usage
```java
Rectangle rect = new Rectangle(4, 5);
System.out.println(rect.getArea());       // 20
System.out.println(rect.getPerimeter());  // 18
```
## Motivation
This project was created to reinforce foundational object-oriented programming concepts such as class design, constructors, and method abstraction.

## Concepts Used
- Object-Oriented Programming (OOP)
- Encapsulation
- Class methods
