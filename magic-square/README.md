# Magic Square Generator (Java)

## Overview
This program generates random 3×3 matrices until it finds a valid magic square, where all rows, columns, and diagonals sum to the same value.

## Features
- Randomized generation of 3×3 matrices
- Validation of magic square conditions
- Ensures numbers 1–9 appear exactly once
- Tracks number of attempts
- Measures execution time

## Example Output
It is a magic square [[8,1,6],[3,5,7],[4,9,2]]<br>
It took 2 seconds to find the magic square

## Motivation
This project was created to explore algorithm design, randomness, and constraint validation. It helped develop problem-solving skills for combinatorial problems.

## Concepts Used
- Arrays (2D matrices)
- Randomization and shuffling
- Algorithm validation
- Performance measurement

## How to Run
1. Compile:
   javac MagicSquare.java
2. Run:
   java MagicSquare
