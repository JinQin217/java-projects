# Magic Square Generator (Java)

## Overview
This program generates random 3×3 matrices until it finds a valid magic square.

A magic square:
- Uses numbers 1–9 exactly once
- Each row, column, and diagonal sums to 15

## Features
- Randomized matrix generation
- Validation of magic square conditions
- Tracks number of attempts
- Measures execution time

## Concepts Used
- Arrays (2D matrix)
- Randomization (shuffle algorithm)
- Algorithm validation
- Performance tracking

## How to Run
1. Compile:
   javac MagicSquare.java
2. Run:
   java MagicSquare
